## 2. Privacy Policy

### 2.1 Information We Collect

**Account Information:**
- Username and email address (for login)
- Age and retirement planning data
- Country (optional)
- Password (encrypted with bcrypt)

**Simulation Data:**
- Financial parameters you enter (income, expenses, wealth)
- Life events and planning scenarios
- Simulation results and projections

**Usage Data:**
- Login timestamps
- Number of simulations run
- Export activity
- Session information

### 2.2 How We Use Your Data

**Service Provision:**
- Authenticate your account
- Save and load your simulations
- Generate personalized projections
- Track usage limits (free tier: 10 simulations/month)

**Product Improvement:**
- Analyze aggregated, anonymized data
- Understand user behavior patterns
- Improve simulation algorithms
- Identify popular features

**Research & Development:**
- Study financial planning behaviors across demographics
- Age-based wealth and income patterns
- Savings rate trends
- Retirement planning statistics

### 2.3 Data Anonymization

**For Analytics & Research:**
Financial amounts are stored as **anonymized brackets**:
- Wealth brackets: €0-25k, €25k-50k, €50k-100k, etc.
- Income brackets: €0-30k, €30k-50k, €50k-75k, etc.
- Property value brackets: €0-100k, €100k-250k, etc.

**What We May Share:**
- ✅ Aggregated statistics (e.g., "average user is 32 years old")
- ✅ Anonymized patterns (e.g., "30-35 year olds save 20% on average")
- ✅ De-identified trends across user cohorts

**What We DON'T Share:**
- ❌ Your name, email, or username
- ❌ Exact financial amounts
- ❌ Personally identifiable information (PII)
- ❌ Individual simulation details

### 2.4 Data Storage & Security

**Storage:**
- Data stored in PostgreSQL database on Render.com
- Passwords encrypted with bcrypt (industry standard)
- Database access restricted to authorized developers only

**Security Measures:**
- Encrypted database connections (SSL/TLS)
- User isolation (each user can only access their own data)
- Regular security updates
- No third-party data sharing

**Data Retention:**
- Account data: Retained until deletion requested
- Simulations: Retained indefinitely for research (anonymized)
- Usage stats: Retained for analytics
- You can request data deletion at any time
