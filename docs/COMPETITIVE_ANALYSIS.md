# FinSim Competitive Analysis

## üèÜ Direct Competitors (Monte Carlo Simulation Tools)

### 1. **ProjectionLab** (https://projectionlab.com)
**Pricing:** $120/year ($10/month)
**Target:** FIRE community, serious planners
**Features:**
- Monte Carlo simulations
- Retirement planning
- Goal tracking
- What-if scenarios
- Net worth tracking

**Strengths:**
- Clean, modern UI
- Very popular in r/financialindependence
- Active development
- Mobile-friendly

**Weaknesses:**
- Expensive for casual users
- US-centric (limited international support)
- No free tier

**Market Position:** Premium product, well-established


### 2. **Firecalc** (https://firecalc.com)
**Pricing:** FREE
**Target:** FIRE community
**Features:**
- Historical backtesting (not true Monte Carlo)
- Retirement withdrawal rates
- Safe withdrawal rate (SWR) calculations
- Simple interface

**Strengths:**
- Completely free
- Trusted in FIRE community (since 1990s)
- Historical data-based

**Weaknesses:**
- Outdated UI
- Limited customization
- US-only data
- No account/saving features

**Market Position:** Legacy free tool, trusted but dated


### 3. **OnTrajectory** (formerly Boldin)
**Pricing:** Free basic, $149/year premium
**Target:** General retirement planners
**Features:**
- Monte Carlo simulations
- Retirement planning
- Social Security optimization
- Tax planning
- Financial advisor matching

**Strengths:**
- Professional grade
- Tax optimization
- Advisor integration

**Weaknesses:**
- Complex for beginners
- Premium tier expensive
- Limited free features

**Market Position:** Professional/advisor-oriented


### 4. **MaxiFi Planner** (by Larry Kotlikoff)
**Pricing:** $100/year (was $1,000+)
**Target:** Sophisticated planners
**Features:**
- Economic-based planning
- Lifetime spending smoothing
- Tax optimization
- Monte Carlo analysis

**Strengths:**
- Developed by economist
- Very sophisticated
- Tax-aware

**Weaknesses:**
- Steep learning curve
- Complex interface
- US-centric

**Market Position:** Academic/sophisticated users

## üéØ FinSim's Competitive Advantages

### 1. **Price Point**
- **Free tier:** 10 simulations (vs competitors: $0-150/year)
- **Paid tier:** $4.99-9.99/month (vs $99-150/year)
- **Value:** 40-75% cheaper than ProjectionLab

### 2. **Monte Carlo Focus**
- True Monte Carlo simulation
- Customizable volatility parameters
- Not just historical backtesting
- Comparable to $100+ tools

### 3. **International Support**
- Multi-currency (EUR, USD, GBP, CHF, CAD, AUD)
- Not US-centric
- Global FIRE community
- Currency conversion built-in

### 4. **Open Source Potential**
- Transparency
- Community contributions
- No vendor lock-in
- Educational value

### 5. **Modern Tech Stack**
- Streamlit = fast iteration
- Clean, simple UI
- No legacy code
- Cloud-native (Render)

### 6. **Privacy-First**
- No account aggregation (no bank login)
- Anonymous data only
- No ads
- User-controlled

### 7. **Flexibility**
- Budget builder integration
- Custom life events
- Property scenarios
- International moves
